## III. 视觉主题

### 主题风格

- **风格**：General Versatile（通用多功能）
- **主题**：浅色主题（白色背景 + 彩色强调）
- **气质**：现代、充满活力、专业、创新、科技感
- **适用场景**：创业路演、产品发布、校赛展示

### 配色方案

| 角色 | HEX | 用途 | RGB |
|------|-----|------|-----|
| **主色（活力橙）** | `#FF6B35` | 标题、重点强调、品牌色 | 255, 107, 53 |
| **辅助色（科技蓝）** | `#004E89` | 背景框架、数据可视化、副标题 | 0, 78, 137 |
| **背景** | `#FFFFFF` | 主要内容背景 | 255, 255, 255 |
| **次级背景** | `#F5F5F5` | 卡片背景、分区背景 | 245, 245, 245 |
| **正文文字** | `#333333` | 主要文字 | 51, 51, 51 |
| **辅助文字** | `#666666` | 注释、说明 | 102, 102, 102 |
| **浅文字** | `#999999` | 脚注、补充 | 153, 153, 153 |
| **成功指示** | `#2ECC71` | 正向数据、优势标记 | 46, 204, 113 |
| **警告指示** | `#E74C3C` | 负向数据、问题标记 | 231, 76, 60 |
| **边框/分割线** | `#DDDDDD` | 卡片边框、分割线 | 221, 221, 221 |

### 配色规则

- **60-30-10 规则**：
  - 60% 白色背景（主体）
  - 30% 活力橙（强调、标题、重点）
  - 10% 科技蓝（框架、数据）
- **文字对比度**：≥ 4.5:1（WCAG AA 标准）
- **每页最多 4 种颜色**

## IV. 字体系统

### 字体方案

**预设**：P1（现代商务/科技风格）

| 角色 | 中文 | 英文 | 备用 |
|------|------|------|------|
| **标题** | Microsoft YaHei | Arial | Helvetica |
| **正文** | Microsoft YaHei | Calibri | Arial |
| **强调** | SimHei | Arial Black | Impact |
| **代码** | - | Consolas | Monaco |

**字体栈**：`"Microsoft YaHei", "Arial", "Helvetica", sans-serif`

### 字体大小层级（投影优化版）

**设计原则**：基于内容密度（中等密度，每页 3-5 项），以 32px 正文为基准

| 级别 | 大小 | 用途 | 行高 |
|------|------|------|------|
| H1 - 封面标题 | 84px | 开场/结尾页大标题 | 1.2 |
| H2 - 页面标题 | 56px | 每页主标题 | 1.3 |
| H3 - 副标题 | 40px | 小标题、重点 | 1.4 |
| **Body - 正文** | **32px** | 主要内容（基准） | **1.5** |
| Annotation - 注释 | 24px | 说明、补充信息 | 1.4 |
| Label - 标签 | 18px | 图表标签、脚注 | 1.3 |
| Caption - 题注 | 14px | 图片说明、版权 | 1.2 |

**投影适配说明**：
- 32px 正文在阶梯教室投影时清晰易读（后排也能看清）
- 所有字体按比例增大，保持视觉层级
- 行高增加，提升可读性

## V. 布局原则

### 页面结构

```
┌─────────────────────────────────────┐
│  Header Area (50px)                 │  标题区域
├─────────────────────────────────────┤
│                                     │
│  Content Area (620px)               │  内容区域
│                                     │
├─────────────────────────────────────┤
│  Footer Area (30px)                 │  页脚区域
└─────────────────────────────────────┘
```

### 6 种布局模式

| 布局 | 适用场景 | PPT 16:9 参考尺寸 |
|------|---------|------------------|
| **单列居中** | 封面、结尾、关键页 | 内容宽 800-1000px，水平居中 |
| **两列** | 对比分析、左图右文 | 列比 1:1 或 3:2，间距 60px |
| **三列** | 并行要点、流程步骤 | 列比 1:1:1，间距 40px |
| **四象限** | 矩阵分析、分类 | 象限 560×250px，间距 30px |
| **上下分割** | 超宽图片 + 文字 | 图片全宽，文字区 ≥150px 高 |
| **左右分割** | 标准/竖图 + 文字 | 图片在侧，文字区 ≥280px 宽 |

### 间距规范

- **页边距**：左右 60px，上下 50px
- **元素间距**：40-60px（大间距）、20-30px（中间距）、10-15px（小间距）
- **卡片内边距**：20px
- **圆角半径**：8px（卡片）、4px（按钮）

## XI. 技术约束提醒

### SVG 生成规则

- ✅ **允许**：inline styles、HEX 颜色、fill-opacity、stroke-opacity、<tspan> 换行
- ❌ **禁止**：clipPath、mask、<style>、class、@font-face、<foreignObject>、<animate>、<script>
- ✅ **图片引用**：`<image href="../images/xxx.png" preserveAspectRatio="xMidYMid slice"/>`
- ✅ **图标占位符**：`<use data-icon="icon-name" x="" y="" width="48" height="48" fill="#HEX"/>`

### PPT 兼容性规则

| 禁止语法 | 正确替代 |
|---------|---------|
| `fill="rgba(255,255,255,0.1)"` | `fill="#FFFFFF" fill-opacity="0.1"` |
| `<g opacity="0.2">...</g>` | 在每个子元素上设置 `fill-opacity` |
| `<image opacity="0.3"/>` | 在图片后叠加 `<rect fill="背景色" opacity="0.7"/>` |
| `marker-end` 箭头 | 用 `<polygon>` 绘制三角形箭头 |
